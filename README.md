# desobe.ir source code
To run it locally:\
cd into the downloaded folder and install dependencies (three and vite) as per Three.js instructions:\
# three.js\
npm install --save three\

# vite\
npm install --save-dev vite\

then:\
npx vite\
and visit \
http://localhost:5173
